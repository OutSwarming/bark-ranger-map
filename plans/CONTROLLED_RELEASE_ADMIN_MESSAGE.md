# Controlled Release Admin Message

Date: 2026-05-09
Audience: admins/mods/VIP testers for controlled release.
Status: draft message for Carter to adapt.

## Short Admin Message

Hi everyone, I am opening a small controlled beta of BARK Ranger Map for a limited group of testers.

The goal is to test the map, sign-in, visited park tracking, search, filters, route tools, leaderboard, feedback, and Premium test checkout flow before any public paid launch.

Important:

- Please do not share the link publicly yet.
- Lemon Squeezy checkout is still in test mode.
- Do not use real payment information unless I directly ask you to.
- Free accounts can track 5 visited parks.
- More than 5 tracked parks and route/saved-route tools are Premium features.
- Coupon codes, if provided, are entered on Lemon checkout.

Please report issues with:

- device,
- browser,
- account email,
- screenshot,
- exact action,
- what happened,
- what you expected.

If you see a blank screen, another user's data, unexpected real payment behavior, or Premium access that looks wrong, stop testing and message me immediately.

Thank you for helping me make this safe and solid before a wider release.

## Longer Admin/Moderator Context

This is not a paid public launch. It is a controlled beta for 25-50 people. I am watching Firebase usage, Lemon test-mode checkout/webhooks, account behavior, route/geocode usage, feedback, and support issues.

Please avoid broad posts or public promotion until I confirm the beta is stable. If testers ask about Premium, tell them Premium checkout is being tested in Lemon test mode and that real paid launch is later.

Known beta expectations:

- small bugs are expected,
- feedback is valuable,
- map/search should work signed out,
- accounts should keep each user's data separate,
- payment/live subscriptions are not live yet,
- app behavior can be paused quickly if needed.
